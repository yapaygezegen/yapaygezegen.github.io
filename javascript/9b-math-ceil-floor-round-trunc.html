JavaScript Math Metodları: ceil() & floor() & round() & trunc()

ceil() Metodu

Ceil: sayıyı yukarıya doğru tam sayıya yuvarlar.

Math.ceil(4.2) // 5
Math.ceil(-7.65) // -7

floor() Metodu

Floor: sayıyı aşağıya doğru tam sayıya yuvarlar.

Math.floor(7.75) // 7
Math.floor(-35.15) // -36

round() Metodu

Round: sayıyı en yakın tam sayıya yuvarlar. Virgülden sonraki değerin 5 ve üzerin olması durumunda yukarıya, altında olması durumunda aşağıya yuvarlar.

Math.round(5.7) // 6

Math.round(5.5) // 6

Math.round(5.3) // 5

trunc() Metodu

Trunc: sayının kesirli basamakların siler ve tam sayı olarak döndürür.

Math.trunc(15.3) // 15

Math.trunc(-53.2) // -53

Math.trunc(-0.28) // -0

Math.trunc(0.35) // 0
