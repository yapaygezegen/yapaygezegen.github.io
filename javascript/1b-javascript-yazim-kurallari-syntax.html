JavaScript Yazım Kuralları (Syntax)

  • Kod bloğu içerisine: tek satırda veya çoklu satırda yorum alanı eklenebilir. Yorum satırları, kodun çalışmasını etkilemez. Geriye dönük düzenlemeler için kod işlevinin hatırlanmasına yardımcı olur. (// Tek satır yorum alanı oluşturur.) (/* Çok satırlı yorum alanı oluşturur. */)
    • // Tek satır yorum alanı oluşturur.

    /*
    Çok
    satırlı
    yorum alanı oluşturur.
    */
  • Değişken atamalarında değişken türü belirtilmez.
  • Kod bloğunun sonunda (;) işareti konulması zorunlu değildir. Kullanılmadığı durumlarda hata vermez.
    • var degiskenAdi1;

    var degiskenAdi2 // ; kullanılmak zorunda değildir
  • Metinsel ifadeler için " " veya ' ' tırnak işaretleri kullanılır.
  • Sayısal ifadeler tırnak işaretleri içinde kullanıldığında veri türü metinsel hale dönüşür.
    • var metin = "Yapay Gezegen"

    var sayi1 = 10 // sayısal değişken

    var sayi2 = "10" // metinsel (string) değişken
  • Girinti bırakmak zorunlu değildir. Kodun daha okunaklı olabilmesi için girinti kullanımı tavsiye edilir.
    • function fonksiyonAdi() {
    console.log("Merhaba JavaScript")
    }

    function fonksiyonAdi() {
    console.log("Merhaba JavaScript")
    }
