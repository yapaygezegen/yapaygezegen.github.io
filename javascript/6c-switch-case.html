JavaScript Switch Case

Switch Case: birden fazla koşulun kontrol edilmesi gereken durumlarda kullanılır. switch ifadesi ile bir değişkenin değerini alır. Bu değere karşılık gelen case bloklarından birini çalıştırır. Hiçbir case'in çalışmadığı durumda default kodları çalışır. * default kullanımı zorunlu değildir.

var gun = "perşembe"

switch (gun) {

case "pazartesi":
console.log("Haftanın 1. günü")
break

case "salı":
console.log("Haftanın 2. günü")
break

case "çarşamba":
console.log("Haftanın 3. günü")
break

case "perşembe":
console.log("Haftanın 4. günü") // eşleşen case bloğu çalışır
break

case "cuma":
console.log("Haftanın 5. günü")
break

case "cumartesi":
console.log("Haftanın 6. günü")
break

case "pazar":
console.log("Haftanın 7. günü")
break

default:
console.log("Hatalı gün değeri girildi.")
break
}
