JavaScript String Metodları: trim() & trimStart() & trimLeft() & trimEnd() & trimRight()

trim() Metodu

trim: string ifadenin başında ve sonunda bulunan boşlukları (space, tab, newline) silmek için kullanılır. * Boşluklarını sildiği string ifadenin kendisi üzerinde değişiklik yapmaz. Bu kural, diğer tüm trim işlemleri için geçerlidir.

let metin1 = " JavaScript Programlama Dili "

metin1.trim() // "JavaScript Programlama Dili"

trimStart() & trimLeft Metodları

trimStart & trimLeft: string ifadenin sadece başında bulunan boşlukları silmek için kullanılır. Bu iki metod aynı işi yapar.

let metin2 = " Sadece soldaki/baştaki boşlukları siler. "

metin2.trimStart() // "Sadece soldaki/baştaki boşlukları siler. "

metin2.trimLeft() // "Sadece soldaki/baştaki boşlukları siler. "

trimEnd() & trimRight() Metodları

trimEnd & trimRight: string ifadenin sadece sonunda bulunan boşlukları silmek için kullanılır. Bu iki metod aynı işi yapar.

let metin3 = " Sadece sağdaki/sondaki boşlukları siler. "

metin3.trimStart() // " Sadece sağdaki/sondaki boşlukları siler."

metin3.trimLeft() // " Sadece sağdaki/sondaki boşlukları siler."
