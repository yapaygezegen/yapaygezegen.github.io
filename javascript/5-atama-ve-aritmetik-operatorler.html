JavaScript Operatörler - 1

Atama Operatörü (Assignment Operator)

Bir değeri bir değişkene atamak için kullanılır. Değişkenin mevcut değerini değiştirmek için de kullanılabilir.

var veri1 = 100 // veri1 değişkeni için 100 değeri atandı

veri1 = 200 // veri1 değişkeninin mevcut 100 değeri, 200 ile değiştirildi

Aritmetik Operatörler

Sayılar üzerinde matematiksel işlemleri gerçekleştirmek için kullanılır. Aritmetik operatörler:

  • + Toplama Operatörü
    • 3 + 5 // 8

    var deger1 = 10

    var deger2 = 20

    var toplamaSonucu = deger1 + deger2

    toplamaSonucu // 30
  • - Çıkarma Operatörü
    • 10 - 3 // 7

    var deger1 = 10

    var deger2 = 20

    var cikarmaSonucu = deger1 - deger2

    cikarmaSonucu // -10
  • * Çarpma Operatörü
    • 12 * 15 // 180

    var deger1 = 7

    var deger2 = 8

    var carpmaSonucu = deger1 * deger2

    carpmaSonucu // 56
  • / Bölme Operatörü
    • 150 / 3 // 50

    var deger1 = 50

    var deger2 = 30

    var bolmeSonucu = deger1 / deger2

    bolmeSonucu // 1.6666666666666667
  • ** Üs Alma Operatörü
    • 2 ** 3 // 8

    var taban = 5

    var us = 3

    var usAlmaSonucu = taban ** us

    usAlmaSonucu // 125
  • % Mod Operatörü (Remainder)
    • 23 % 5 // 3

    var sayi = 153

    var modu = 20

    var modSonucu = sayi % modu

    modSonucu // 13
  • ++ Arttırma Operatörü
    • 10++ // 11

    var artmak = 50

    artmak++ // 51
  • -- Azaltma Operatörü
    • 10-- // 9

    var azaltmak = 40

    azaltmak-- // 39

    * Aritmetik operatörler, atama operatörü ile birleştirilebilir.

    var deger1 = 10

    var deger2 = 20

    deger1 += deger2 // + aritmetik operatörü ile = atama operatörü birleştirildi

    deger1 // 30 -> bütün aritmetik işlemler atama operatörü ile birleştirilebilir (-, *, /, **, %, ++, --)

    var deger3 = 30

    var deger4 = 40

    deger4 *= deger3 // * aritmetik operatörü ile = atama operatörü birleştirildi

    deger4 // 1200
