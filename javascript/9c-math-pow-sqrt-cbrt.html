JavaScript Math Metodları: pow() & sqrt() & cbrt()

pow() Metodu

Pow: sayının üssünü alır. İlk parametre taban, ikinci parametre üs değeridir.

Math.pow(2,3) // 8

Math.pow(7,0) // 1

Math.pow(4,0.5) // 2

Math.pow(16,-1) // 0.0625

Üs alma metodunu ** operatörünü kullanarak da gerçekleştirebiliriz.

2 ** 3 // 8
7 ** 0 // 1
4 ** 0.5 // 2
16 ** -1 // 0.0625

sqrt() Metodu

Sqrt: sayının karekökünü alır.

Math.sqrt(144) // 12

Math.sqrt(10) // 3.1622776601683795

cbrt() Metodu

Cbrt: sayının küpkökünü alır.

Math.cbrt(8) // 2

Math.cbrt(27) // 3

** operatörünü kullanarak: sqrt ve cbrt metodlarını yazababiliriz. Kök alma değerini: 2 (karekök) ve 3 (küpkök) değerlerinden daha fazla olabilir.

// karekök için (1/2)
144 ** (1/2) // 12
10 ** (1/2) // 3.162277660168379

//küpkök için (1/3)
8 ** (1/3) // 2
27 ** (1/3) // 3

// (1 / istenilen kök alma değeri)

16 ** (1/4) // 2
