JavaScript String Metodları: concat()

concat() Metodu

Concat: iki veya daha fazla string ifadeyi birleştirmek için kullanılır. Mevcut stringleri değiştirmeden, yeni bir string döner. Birleştirilen ifadelerin arasına: boşluk, noktalama işareti veya başka karakterlerin eklenmesine imkan verir.

let kelime1 = "Merhaba"

let kelime2 = "JavaScript"

let kelime3 = "Dünyası"

let sonuc = kelime1.concat(" ", kelime2, " ", kelime3)

console.log(sonuc) // Merhaba JavaScript Dünyası

String ifadeleri birleştirmek için "+" operatörü, en yaygın kullanılan yöntemdir. İki veya fazla string ifadeyi birleştirebilir. String üzerinde işlem yapılacak ise: template literals tercih edilir. (Template literals kullanımı İlkel Veri Tipleri bölümünde anlatılmıştır.)

Örnek: elimizde aynı soyadını taşıyan kişilerin listesi olsun. Dizi içerisinde bu kişilerin sadece adlarını saklayalım. "+" operatörü ve for of döngüsünü kullanarak: adları ve soyadı string ifadelerini birleştirelim. Birleşim sonuçlarını konsol ekranına yazalım.

let adListesi = ["Ayhan","Birkan","Cansu","Deniz","Ezgi","Gökhan"]

let soyad = "Kurt"

for (const adlar of adListesi) {
console.log(adlar + " " + soyad);
}

// Ayhan Kurt
// Birkan Kurt
// Cansu Kurt
// Deniz Kurt
// Ezgi Kurt
// Gökhan Kurt
