JavaScript String Metodları: startsWidth() & endsWith()

startsWidth() Metodu

startsWidth: belirtilen başlangıç index değerinden sonra, istenilen ifade ile başlayıp başlamadığını kontrol eder. Başlıyorsa true, başlamıyorsa false döner. Başlangıç index değeri varsayılan olarak sıfırdır. Büyük küçük harflere duyarlıdır.

let metin = "JavaScript Dersleri"

metin.startsWith("JavaSc") // true

metin.startsWith("JAvaSc") // false

metin.startsWith("Ders",11) // true

metin.startsWith("Ders",10) // false

endsWith() Metodu

endsWith: belitrilen bitiş index değerinden önce, istenilen ifade ile sonlanıp sonlanmadığını kontrol eder. Bitiyorsa true, bitmiyorsa false döner. Bitiş index değeri varsayılan olarak string ifadenin length değeri kadardır. Büyük küçük harflere duyarlıdır.

var metin = "Yapay Gezegen"

metin.endsWith("Gezegen") // true

metin.endsWith("Geze") // false

metin.endsWith("egen") // true

metin.endsWith("Gezegen",14) // true

mmetin.endsWith("Gezegen",12) // false

startsWidth ve endsWith metodları: string manipülasyonları, veri doğrulama ve filtreleme işlemleri gibi alanlarda sıkça kullanılır.
